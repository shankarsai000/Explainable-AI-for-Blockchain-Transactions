## Instructions

1. Install Jupyter Notebook:
   ```bash
   pip install jupyter notebook
   ```

2. Start Jupyter:
   ```bash
   jupyter notebook
   ```

3. Open the desired notebook to view the training pipeline, feature engineering, and evaluation metrics.
